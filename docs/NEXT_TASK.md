# 下一個任務：音程練習室

目前下一個主要開發任務是新增「音程練習室」。

請注意：這個任務只新增音程練習室 UI、練習資料生成、五線譜顯示與完成流程。不要改調音器核心，不要加入自動判斷。

## 功能定位

音程練習室 = 看譜練習 + 調性內音程移動 + 自我判斷 + 節拍引導。

不是：

- 音準判分機。
- 耳訓考試。
- 調音器偵測遊戲。
- 自動評分系統。

明確不要做：

- 不要用麥克風判斷是否吹對。
- 不要顯示吹對 / 吹錯。
- 不要偵測第一音與第二音。
- 不要判斷 cents 是否準。
- 不要計算命中率。
- 不要做音準分數。
- 不要顯示即時調音器。
- 不要改動調音器核心。

## 入口需求

開始練習頁中的音程練習室卡片改為已開放，不再顯示準備中。

卡片文案：

- 標題：音程練習室
- 副標：從二度到八度，跟著五線譜練習不同調性的音程移動。

使用者先前也要求：

- 按鍵練習室與音階練習室位置對調。
- 原「音階練習室」改名為「音程練習室」。

接手時請先確認目前頁面狀態，再小範圍調整。

## 練習內容生成

依照使用者選擇的調性與音程，自動產生調性內音程練習。

範例：

### C 大調 + 上行三度

- C4 → E4
- D4 → F4
- E4 → G4
- F4 → A4
- G4 → B4
- A4 → C5
- B4 → D5
- C5 → E5

### G 大調 + 上行三度

- G4 → B4
- A4 → C5
- B4 → D5
- C5 → E5
- D5 → F#5
- E5 → G5
- F#5 → A5
- G5 → B5

### F 大調 + 上行三度

- F4 → A4
- G4 → Bb4
- A4 → C5
- Bb4 → D5
- C5 → E5
- D5 → F5
- E5 → G5
- F5 → A5

注意：

- 要依照調性產生正確升降記號。
- 不要全部固定 C 大調。
- 不要把 Bb 顯示成 A#，除非該調性需要。
- 不要把 F# 顯示成 Gb，除非該調性需要。
- 第一版先以高音譜號為主。
- 音域建議控制在 C4-C6，或目前 App 適合半音階口琴初學者的舒適音域。

## 五線譜顯示

音程練習室重點是五線譜。

技術方案：

- 優先考慮 VexFlow。
- 如果引入套件成本太高，第一版可使用自製 SVG 五線譜。

顯示要求：

- 高音譜號。
- 調號。
- 音符。
- 小節線。
- 每頁最多顯示 4 組音程。
- 手機畫面可讀。
- 不要讓五線譜超出手機畫面。

## 手機版布局

畫面結構：

- 上方：`C 大調｜上行三度`
- 中間：五線譜卡片。
- 五線譜卡片中顯示目前 1-4 組音程。
- 下方：目前音程文字輔助。
- 底部操作：
  - 上一頁
  - 下一頁
  - 完成一循環
  - 重新開始

不要做桌機版大排版。不要一次塞完整長譜。

## 分頁規則

每頁顯示 4 組音程。

如果該練習共有 8 組，分 2 頁。

例如 C 大調三度：

第 1 頁：

- C4 → E4
- D4 → F4
- E4 → G4
- F4 → A4

第 2 頁：

- G4 → B4
- A4 → C5
- B4 → D5
- C5 → E5

## 練習流程

1. 點擊音程練習室。
2. 選擇調性。
3. 選擇音程。
4. 選擇方向。
5. 選擇 BPM。
6. 選擇循環數。
7. 點擊開始練習。
8. App 顯示五線譜。
9. 使用者跟著譜自行練習。
10. 使用者完成一輪後按「完成一循環」。
11. 循環數累加。
12. 達到設定循環數後，顯示完成畫面。
13. 發放水滴。

## 節拍與水滴

- 第一版節拍器可以持續播放即可。
- App 不強制自動換頁。
- 使用者自己按下一頁或完成一循環。
- 完成目前音程練習的一輪 = 1 循環。
- 完成 1 循環 = 1 滴。
- 單次最高 8 滴。
- 每日練習水滴仍受 80 滴上限限制。
- 沿用現有水滴系統，不要重寫。

## 完成畫面

練習完成後顯示：

- 標題：音程練習完成！
- 調性：C 大調
- 音程：三度
- 方向：上行
- 完成循環：4 / 4
- 獲得水滴：4 滴

## 資料紀錄

可新增音程練習紀錄，但不要破壞舊資料。

建議紀錄格式：

```js
intervalPracticeHistory = {
  date: "YYYY-MM-DD",
  key: "C",
  mode: "major",
  interval: "third",
  direction: "ascending",
  cyclesCompleted: 4,
  waterEarned: 4
}
```

如果目前已有統一 practice history，請整合進現有格式。

## 驗收清單

- 音程練習室入口已開放。
- 設定可選大調、音程、方向、BPM、循環。
- 會依調性產生正確音名與升降記號。
- 五線譜手機可讀。
- 每頁最多 4 組。
- 上一頁 / 下一頁可用。
- 完成一循環可累加。
- 達成循環數後顯示完成畫面。
- 完成後發放水滴並尊重每日 80 滴練習上限。
- 沒有接入 mic / detector / cents 判斷。
- 沒有改調音器核心。

## 其他排隊待辦

音程練習室是主線。其他可後續處理：

- 初始選擇視窗後兩隻植物大小與前三隻統一。
- 首頁開始練習按鈕圖片背景細節與白邊調整。
- 雨天事件 50% 機率實測與動畫確認。
- 菇鳴靈、花樂仙子、幸葉靈等去背細節最終確認。
- PWA / Capacitor 打包準備項目補齊。
