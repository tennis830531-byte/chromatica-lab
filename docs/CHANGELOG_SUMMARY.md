# 最近重要變更摘要

這份摘要整理近期已完成或已大幅整理過的功能，方便新聊天室快速接手。

## 首頁與快速入口

- 首頁 hero 視覺多次調整，維持暖米色與日系手帳感。
- 首頁品牌文字修正為 `Chromatic Harmonica Lab`。
- 首頁主按鈕改為「開始練習~」，使用圖片背景與主入口互動效果。
- 首頁展示植物加入背景舞台與植物名稱小卡。
- 首頁展示植物會左右搖擺與跳躍。
- 點擊首頁展示植物時只播放跳躍動作，不跳轉花園。
- 快速入口新增全球排行榜入口。
- 快速入口保留音符地圖、全音域調音器、開始練習、每日目標、精靈花園、全球排行榜、設定。
- 快速入口逐步調整為更像手帳貼紙入口，弱化或移除傳統卡片感。

## 設定頁

- 設定頁整理為多個卡片區塊：
  - 收音設定
  - 練習設定
  - 聲音設定
  - 顯示與動畫
  - 通知與語言
  - 會員帳號
  - 關於
- 設定卡片改為可收合，避免一次顯示太多資訊。
- Google 登入 UI 預留完成，沒有做假登入。
- 聲音設定已連動既有 sound manager 的開關資料。
- 花園頁面音樂預設改為開。

## 字體與文案

- 加入 Google Fonts：
  - LXGW WenKai TC
  - Noto Sans TC
- 建立字體變數：
  - 標題與角色名稱使用手帳感字體。
  - 內文使用 Noto Sans TC。
  - 數字使用 system-ui 與 tabular nums。
- 整理部分標題與副標層級，減少重複文案。
- 花園文案改為「口琴植物精靈」相關語氣。

## 全音域調音器

- 調音器卡片視覺改為深綠調音背景圖。
- 修正調音器圖案在深背景下不清楚的問題，加入合適底色。
- 新增 A4 調音基準說明卡片。
- 說明內容包含：
  - `A = 440 Hz` 指國際標準術語的 A4。
  - 簧片調音先用 442 Hz。
  - 最後判斷用 440 Hz 檢查實際吹出來的音，並看裝回琴後、用正常力度吹的結果。

## 開始練習頁

- 長音練習室保持已開放。
- 「已開放」徽章改成與底部練習入口一致的紅色系。
- 按鍵練習室與原音階練習室位置有調整需求。
- 原音階練習室規劃改為音程練習室。
- 音程練習室是下一個主要待辦。

## 長音練習室

- 長音練習室保留原有偵測核心。
- 練習完成後依循環給水滴。
- 練習完成 toast 可分行顯示：
  - 今日連續學習目標達成狀態。
  - 獲得水滴。
  - 特殊事件或里程碑 bonus。
- 長音練習教學 modal 已做過精簡，移除部分冗餘說明卡片。

## 每日目標與連續練習

- 每日目標頁整理過版面與文案。
- 完成當日所有任務會額外獲得 20 滴水滴。
- 每日任務規則調整方向：
  - 每一類練習需完成不同組合兩次才算該項任務完成。
  - 例如平穩長音、音量分層、漸強、漸弱、山形各自是任務類型。
- 練習項目完成每日任務後可顯示綠框與角落打勾。
- 連續學習 freeze 規則改為連續學習 3 天獲得 1 張，最多 2 張。

## 精靈花園水滴系統

- 每日首次進入 App bonus：
  - 每日 5 滴。
  - 不計入每日練習上限。
  - 不推進 streak。
- 每日練習水滴：
  - 1 循環 = 1 滴。
  - 單次最高 8 滴。
  - 每日練習水滴上限 80 滴。
  - 80 滴只限制當天透過練習獲得的水滴，不限制庫存總量。
- 完成所有每日任務：
  - 額外 20 滴。
- 連續練習里程碑：
  - 7 天 50 滴。
  - 30 天 100 滴。
  - 60 天 120 滴。
  - 90 天 150 滴。
  - 120 天後每 30 天 150 滴。
- 雨天事件：
  - 已改為進入精靈花園頁面時抽選。
  - 每天最多抽一次。
  - 近期需求改為 50% 機率。
  - 動畫後再顯示提示。

## 植物成長

- 植物成熟需求提高為總計 530 滴。
- 階段需求：
  - Stage 1 → Stage 2：100 滴。
  - Stage 2 → Stage 3：180 滴。
  - Stage 3 → 成熟：250 滴。
- 階段 UI 顯示階段進度與今日練習水滴。
- 移除不必要的「還需要 X 滴」與總成長行顯示。
- 單次澆水不應跳過多階段，需要逐階段進化。

## 植物精靈與圖鑑

- 目前整理前五種植物精靈：
  - 旋律芽芽 → 旋律葉靈 → 旋律森使
  - 菇鳴靈 → 菇鳴樂手 → 菇鳴賢者
  - 花樂精靈 → 花樂舞靈 → 花樂仙子
  - 幸葉芽 → 幸葉靈 → 幸葉祭司
  - 竹芽童 → 竹音童 → 竹音翁
- 圖鑑標題調整為「植物精靈圖鑑」。
- 圖鑑格位固定，不隨隨機初始植物改變。
- 未種植或未解鎖植物使用問號圖。
- 圖鑑 modal 階段名稱改為各階段專屬名稱。
- 點選圖鑑不同階段時，modal 標題需跟著更新。
- 多次調整植物在圖鑑、首頁與花園舞台上的比例與底部對齊。
- 已新增第五種植物「竹音童」相關素材與去背流程。

## 初始植物選擇

- 新增第一次進入花園時選擇初始植物精靈的流程。
- 可選前五隻植物精靈。
- 使用者必須選擇一隻作為第一株培養植物。
- 「再看看」應回首頁，不應進入空白花園。
- 點選植物卡片時，選中的植物要左右搖擺與跳躍；切換選擇時取消前一隻動作。
- 點「就決定是你了！」後播放加入動畫與音效，再顯示初始夥伴加入訊息。
